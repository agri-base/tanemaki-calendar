# 🌱 種まきカレンダー

あなたの地域で、今、種をまける野菜を見つけよう。

## 特徴

- 📍 **郵便番号で自動判定** - 地域の気候に合った種まき時期を表示
- 🌙 **月齢カレンダー連携** - 自然農で重視される種まき好適日を自動判定
- ☀️ **天気予報連携** - 今週末の種まき最適日を提案
- 📦 **種リスト管理** - 買った種を一元管理
- 🔔 **リマインダー機能** - 種まき適期を通知
- 🌿 **自然農/慣行農業対応** - 栽培方法を切り替え可能
- 🌾 **50種の野菜データベース** - 固定種・在来種の情報も充実

## デプロイ方法

### Vercelで公開（推奨）

1. このリポジトリをGitHubにプッシュ
2. [Vercel](https://vercel.com)にアクセス
3. "Import Project"でリポジトリを選択
4. デプロイ完了！

### ローカルで実行

```bash
npm install
npm run dev
```

http://localhost:3000 でアクセス
